
# Sistema Escuela Java

Proyecto desarrollado en Java con JDBC y MySQL.

## Funcionalidades

- CRUD de alumnos
- CRUD de maestros
- CRUD de director
- CRUD de materias
- CRUD de calificaciones

## Tecnologías

- Java
- MySQL
- JDBC
- POO

## Estructura

- src → código fuente
- recursos → base de datos y diagramas

## Ejecución

1. Crear la base de datos ejecutando el script:
   recursos/base_de_datos.sql

2. Configurar la conexión en:
   Conexion.java

3. Ejecutar la clase principal:
   Escuela_Proyecto.java
   
## Patrón utilizado

- DAO (Data Access Object)
