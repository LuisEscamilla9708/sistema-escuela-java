
# Sistema Escuela Java

Proyecto desarrollado en Java con JDBC y MySQL.

## Funcionalidades

- CRUD de alumnos
- CRUD de maestros
- CRUD de director
- CRUD de materias
- CRUD de calificaciones

## Tecnologías

- Java
- MySQL
- JDBC
- POO

## Estructura

- src → código fuente
- recursos → base de datos y diagramas
